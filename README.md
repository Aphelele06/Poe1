3. INTRODUCTION
This project is a Cybersecurity Awareness Bot developed using C# Windows Presentation Foundation (WPF) and MySQL.
The system provides an interactive chatbot, task management system, quiz module, and activity logging features.
4. PROBLEM STATEMENT
Many users lack awareness of cybersecurity risks. This system aims to educate users through interactive learning and task-based engagement.
5. OBJECTIVES
Develop a chatbot using natural language commands
Implement task management with database storage
Create a cybersecurity quiz system
Track user activity using logs
Store all data using MySQL database
6. SYSTEM DESCRIPTION
The system contains:
Chatbot interface for user interaction
Task assistant (Add, Complete, Delete, Show tasks)
Quiz system for cybersecurity learning
Activity log for tracking actions
MySQL database for persistent storage
7. TOOLS & TECHNOLOGIES
C# WPF
MySQL Database
Visual Studio
.NET Framework

Chatbot commands

All features operate successfully without runtime errors.
9. CONCLUSION
The Cybersecurity Awareness Bot successfully demonstrates a functional, interactive system with database integration and user engagement features.
